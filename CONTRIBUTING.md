# コントリビューションガイド

## フィードバックの送り方

### AIに任せる（推奨）

AIに「GitHubに報告して」と言うだけでOKです。

```
「image_urlで画像が表示されなかった。GitHubに報告して」
→ report/report_discussion.py が自動実行されてDiscussionsに投稿されます
```

### 直接投稿する

https://github.com/icloudaichi/utage-skill/discussions/new

カテゴリの目安:
- **Bug**: APIが仕様と異なる動作をする
- **Ideas**: SKILL.mdに追加すべき知見がある
- **Q&A**: 動作が不明・要検証

## スキルファイルの更新提案

SKILL.mdの内容を修正・追加したい場合:

1. このリポジトリをFork
2. 変更をコミット
3. Pull Requestを送る
